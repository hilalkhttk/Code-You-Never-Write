# Code You Never Write — Hilal Khan

Four real-world AI-assisted coding projects, built by describing each problem clearly to an AI
tool and verifying every result against a fact I already knew was true — never trusting output
that couldn't be checked.

**AI tool used:** Claude (claude.ai)

## Projects

| Folder | Project | What it does |
|---|---|---|
| `project-1-money-detective/` | Money Detective | Finds recurring charges and duplicate/forgotten subscriptions in transaction history |
| `project-2-whats-my-grade/` | What's My Grade, Really | Calculates true current grade using my teacher's actual weighting/drop rules, and the final-exam score needed for a target grade |
| `project-3-books-dont-match/` | The Books Don't Match | Reconciles a known hand-counted total against a messy digital payment record |
| `project-4-organize-the-mess/` | Organize the Mess | Safely finds duplicate/large files and organizes a messy folder by type — copy-only, dry-run first |

Each folder contains:
- The final working script
- A README explaining the problem, verification, and result
- `prompts.md` — the prompts used to build the script
- Sample/demo data (replaced with real data before final submission, sanitized where needed)

## Report
See `REPORT.md` for the full write-up covering all four projects: prompts used, verification
method, problems faced, and final results.

## Note on Data
Sample/demo data is included in this repo to show each script works. Before submitting, real
data should be swapped in — with any sensitive values (real account numbers, passwords, etc.)
replaced with dummy values, per the assignment's privacy requirement.
